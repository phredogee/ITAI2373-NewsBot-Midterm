# ITAI2373-NewsBot-Midterm
This repository contains the NLP pipeline and classification files for the NewsBot midterm project, focusing on processing and categorizing raw news data.

---

### Key Components
* **Data Source:** Uses a [BBC news dataset](https://github.com/phredogee/ITAI2373-NewsBot-Midterm/blob/main/bbc-news-data.csv) containing raw articles.
* **Core Modules:**
    * **Preprocessing:** Text cleaning, tokenization, stop word removal, and lemmatization.
    * **Feature Extraction:** TF-IDF vectorization and POS tagging (NLTK/spaCy).
    * **Analysis:** Dependency parsing, sentiment classification, and Named Entity Recognition (NER).
    * **Machine Learning:** Training and evaluation of Naive Bayes, SVM, and Logistic Regression models.

### Project Workflow
The system follows a modular pipeline from raw text ingestion to an aggregated "NewsBot Output" that includes visualizations and insight reports.
